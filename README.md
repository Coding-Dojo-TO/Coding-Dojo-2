# Coding-Dojo-2

# II Toronto Coding Dojo Challenge (JavaScript)

## The Challenge

Write a function that takes two integers, and return their (LCM) Least Commom Multiple

**i.e:**

**lcm(20, 8)**

**Return: 40**

**lcm(15, 9)**

**Return: 45**
